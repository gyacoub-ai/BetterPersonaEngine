Data Integrity & Anti-Hallucination Statement

Every data point in this hub is derived from Better's first-party funded loan data (Q1 2026, n=1,817) and Betsy AI agent conversation transcripts. No external data, assumptions, or AI-generated estimates are used.

Confidence levels are based on statistical margin of error at 95% confidence interval:

HIGH n >= 200Margin of error < 7% - patterns are statistically robust
MODERATE n = 100-199Margin of error 7-10% - patterns are reliable but validate before scaling
DIRECTIONAL n = 50-99Margin of error 10-14% - use for initial framing, not activation
CAUTION n < 50Margin of error > 14% - insufficient for confident conclusions

Transcript theme analysis uses exact string matching (keyword search) against conversation text - not AI summarization or interpretation. Each citation shows the exact count of transcripts containing the theme keywords out of total transcripts analyzed. Themes with fewer than 15% prevalence are excluded.
